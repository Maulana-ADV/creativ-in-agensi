💰 Harga Paket

Mulai dari: IDR 200K

*Harga menyesuaikan dengan jenis kebutuhan media (Desain Logo, Poster, Banner, Merchandising) serta kerumitan konsep visual.

✅ Persyaratan & Ketentuan

  • DP: Pembayaran uang muka (down payment) minimal sebesar 50% untuk memulai tahap penggalian ide kreatif.
  • Waktu: Klien diwajibkan memberikan brief acuan desain secara lengkap dan jelas di awal diskusi projek.
  • Revisi: Bebas mengajukan maksimal 3 kali revisi minor setelah draf rancangan awal diserahkan.
  • Hak Cipta: Hak kepemilikan penuh dan file master final diserahkan seutuhnya kepada klien setelah pelunasan biaya.

🎁 Termasuk dalam Paket

  • Rancangan konsep desain orisinal yang kreatif dan fungsional.
  • Pilihan alternatif pilihan skema warna (Color Palette) dan tipografi (*font*).
  • Master file resolusi tinggi format siap cetak/digital (.AI / .CDR / .PSD / .PDF).
  • *Bonus:* Presentasi visual desain dalam bentuk mockup produk 3D komersial.
